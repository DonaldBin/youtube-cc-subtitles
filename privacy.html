隐私政策 - youtube-CC-subtitles

最后更新日期：2026年4月3日

1. 数据收集声明

本扩展不收集任何个人数据或用户信息。所有功能均在用户浏览器本地运行。

2. 数据使用说明

  • 不收集、存储或传输用户个人信息
  • 不上传任何数据到远程服务器
  • 仅使用 chrome.storage.local 保存字体大小设置（本地存储，仅一个数值）
  • 所有代码在浏览器本地执行，不涉及网络请求

3. 权限使用说明

  • activeTab：仅在用户点击扩展图标时访问当前标签页，用于检测 YouTube 页面
  • host permissions：仅限于 *.youtube.com 域名，用于修改 CC 字幕 CSS 样式
  • storage：仅保存用户设置的字体大小数值（32px-96px 范围）

4. 第三方服务

本扩展不使用任何第三方分析工具、广告服务或外部 API。
